# Soundscope

Soundscope is a music recording software that can display waveforms and recognize audio for analysis.

The project strictly follows architecture specified by [Clean Architecture](https://blog.cleancoder.com/uncle-bob/2012/08/13/the-clean-architecture.html) to achieve separation of concerns.

## Use cases

- Save/Load Audio
- Record Audio
- Playback
- Display Waveform
- Fingerprint Audio with [Chromaprint](https://acoustid.org/chromaprint)
- Identify Song using [Acoustid](https://acoustid.org/)

## API Used

- [Acoustid](https://acoustid.org/)
